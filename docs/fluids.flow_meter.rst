Orifice plates, flow nozzles, Venturi tubes, cone and wedge meters (fluids.flow_meter)
======================================================================================

    .. meta::
      :description: Rigorous set of correlations and solvers for working with flow meters - Orifice plates, nozzles, venturi tubes, cone meters, and wedge meters are all included. Limited two-phase metering support is included. All methods are designed for rating and sizing purposes.
